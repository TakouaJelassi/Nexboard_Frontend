What's included

Built end-to-end

Kanban boards

Four columns — To Do, In Progress, Review, Done. Drag tasks between columns or use the detail view to update status.

Task management

Set priority, due date, assignee and reviewer per task. Filter by priority or due date directly on the board.

Team members

Invite colleagues to a board by email. Member avatars are shown on the board header and task cards.

Token authentication

Register, log in, and stay authenticated with DRF token auth. Guest mode available without an account.

Dashboard overview

See all your boards, open tasks, and a status breakdown chart in one view. Quick access to tasks assigned to you.

REST API

Django REST Framework backend with OpenAPI docs at /api/docs/. Fully browsable and documented.

About

A portfolio project

NexBoard was built as a full-stack portfolio piece by Takoua Jelassi.
The backend is a Django REST API deployed on Render. The frontend is plain HTML, CSS and JavaScript — no framework, no build step.

The goal was to build something complete and functional: authentication, data relationships, a real UI, and a clean deployment.
