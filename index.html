What it does

A small board app with real app basics

Kanban boards

Create boards with To Do, In Progress, Review and Done columns. Tasks can be moved on the board or updated from the detail view.

Task management

Each task can have a priority, due date, assignee and reviewer, so the board has enough structure without becoming heavy.

Team members

Add people to a board by email and see members directly in the board header and on task cards.

Token authentication

Registration and login use DRF token authentication. There is also a guest mode for quickly checking the UI.

Dashboard overview

The dashboard groups boards, open tasks and status numbers in one place, with quick access to assigned work.

REST API

The backend exposes the main project data through Django REST Framework, with OpenAPI docs at /api/docs/.
